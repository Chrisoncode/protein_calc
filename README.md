# Protein Calculator 

A simple, interactive Python program that helps you calculate protein content in various foods. The calculator comes with a built-in database of common protein sources and allows users to add custom foods.

## Features 

- **Extensive Database**: Pre-loaded with 40+ common protein sources across multiple categories
- **Protein Calculation**: Calculate protein content based on food weight in grams
- **Custom Food Addition**: Add your own foods to the database during runtime
- **Organized Display**: View all foods grouped by category with their protein content
- **User-Friendly Interface**: Simple menu-driven interface with input validation
- **Real-time Results**: Instant protein calculations with detailed breakdowns

## Categories 

The calculator includes foods from these categories:
- Meats (chicken, beef, pork, lamb)
- Fish & Seafood (salmon, tuna, shrimp, etc.)
- Eggs & Dairy
- Plant-based proteins (tofu, lentils, beans, etc.)
- Nuts & Seeds
- Protein Supplements
